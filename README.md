# Blackjack Tkinter

A simple Blackjack game built with Python and Tkinter.

## Features

* Graphical User Interface (GUI)
* Hit and Stand buttons
* Dealer AI
* Automatic Ace handling
* Win, Lose, and Draw detection
* New Game button

## Requirements

* Python 3.x
* Tkinter (included with Python)

## How to Run

```bash
python main.py
```
